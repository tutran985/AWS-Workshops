## 1: Cài đặt thư viện Amplify
1. Bạn sẽ cần hai thư viện Amplify cho dự án của mình. 
Thư viện chính `aws-amplify` chứa tất cả các API phía client để kết nối frontend của ứng dụng của bạn với backend và 
thư viện `@aws-amplify/ui-react` chứa các thành phần giao diện người dùng cụ thể cho framework.
- Mở một cửa sổ terminal mới, điều hướng đến thư mục dự án của bạn (`new-react-app`),
 và chạy lệnh sau để cài đặt các thư viện này vào thư mục gốc của dự án.
```bash copy
  npm install aws-amplify @aws-amplify/ui-react 
```
